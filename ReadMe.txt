*Проект* представляет собой базу данных, состоящих из нескольких таблиц, форму, которая должна быть в нескольких вариантах и инструменты для аналитической обработки и 
анализа данных

Несколько таблиц и форм нужны изза того, что приложение должно собирать данные от нескольких исследовательских групп
